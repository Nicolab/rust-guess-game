# Guess number Game in Rust

ʘ‿ʘ ▲ A little guessing game developed in Rust for fun and discovery ▼ ʘ‿ʘ

![Rust lang](rust-logo.png)

## Install the game

Download and compile the source:

```sh
cargo build --release
```

or play direct:

```sh
cargo run
```

Play the game in the terminal (｡◕‿◕｡)
